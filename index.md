# Claude on Microsoft Foundry

Set up and use Anthropic's Claude models through Microsoft Foundry (Azure AI Foundry) — from Azure resource provisioning to agentic coding in VS Code.

## Quick Start

1. [Check the prerequisites](docs/prerequisites.md) for your Azure subscription
2. [Create a Foundry project and deploy a Claude model](docs/azure-setup.md)
3. [Configure authentication](docs/authentication.md) (API key or Entra ID)
4. [Set up VS Code](docs/vscode-setup.md) with Claude Code or Foundry Toolkit
5. [Call the API](docs/api-access.md) from your applications

## What You Get

- **Claude model family** — Opus, Sonnet, and Haiku models deployed as serverless APIs in your Azure environment
- **Enterprise governance** — Azure RBAC, Entra ID authentication, and Azure Monitor integration
- **Agentic coding** — Claude Code in VS Code connected to your own Foundry resource
- **Pay-as-you-go billing** — No upfront commitments, billed through your Azure subscription

## Supported Regions

Claude models on Microsoft Foundry are currently available in **East US 2** and **Sweden Central**.
